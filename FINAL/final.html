NFL Infomap (updated as of April 2023)

About the map

This map contains basic information about each NFL team. The United States shapefile was provided by the US Census Bureau, and the information in the Win Percentage Layer was provided by a Reddit user. There are two baselayers, a gray basemap and dark basemap.

The "Basic Information" layer contains information about each team. Any area that has a team is filled in with team's primary color. Clicking on a filled in polygon opens a popup containing the team's name, logo, year founded, and a link to more information about the team.

The "Superbowl Wins" layer has the number of Superbowls each team has won. Teams filled in with a darker polygon have more wins. Clicking on a polygon creates a popup with the team name and number of Superbowl wins.

The "Win Percentage" layer shows the win percentage of each team. The smaller a cirlce, the closer to 0 a team's win percentage is. Hovering the mouse over a cirlce will create a popup that contains the team name, total number of wins, total number of loses, and total number of ties. The win percentage is at the bottom of the popup.
